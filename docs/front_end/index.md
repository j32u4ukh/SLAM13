# 前端

* 兩視圖幾何(Two-view geometry)

核心問題:**如何根據影像，估計相機運動？**

將特徵點看作固定在 3D 空間的不動點，根據它們在相機中的投影位置，透過最小化『重投影誤差(Reprojection error)』來最佳化相機運動。
